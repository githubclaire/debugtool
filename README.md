# debugtool

print help info
````
help/?
````

Quit
````
quit
````

Print temperature
````
ts
````

Print Vcore
````
vcore
````

Read/Write clock
````
clock d/e/v
````

MIU test connection
````
memtest
````

Program fw to spi flash
````
prog xxx.rom/xxx.bin
````

Dump data from spi flash
````
dump xxx.rom/xxx.bin
````

Read/Write register
````
mmio xxxx
````

I2c read/write data
````
i2c index addr subaddr data
````

Print all info(pcie info/mem info/vcore/clk/temp)
````
print
````
